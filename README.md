# Dataset Twitter/X Sentiment Program Makan Bergizi Gratis (MBG) – 2025

## Deskripsi Umum
Repository ini berisi **dataset hasil crawling percakapan publik** dari platform **Twitter/X** yang berkaitan dengan **Program Makan Bergizi Gratis (MBG)** di Indonesia.

Dataset dikumpulkan **khusus untuk kepentingan akademik dan penelitian**, serta digunakan sebagai data empiris dalam penyusunan tugas akhir/skripsi.

---

## Periode Pengambilan Data
- **Tanggal Mulai**   : 1 Januari 2025  
- **Tanggal Akhir**   : 31 Desember 2025  
- **Rentang Waktu**   : 1 tahun penuh (longitudinal)

Periode ini dipilih untuk menangkap dinamika sentimen publik secara komprehensif sepanjang tahun 2025.

---

## Metode Pengumpulan Data
- Teknik: **Web Crawling / API-based Crawling**
- Sumber data: **Konten publik Twitter/X**
- Jenis data:
  - Teks tweet
  - Waktu publikasi
  - Metadata non-pribadi (tanpa data sensitif)

⚠️ **Catatan penting**:  
Dataset **tidak mengandung data privat**, tidak memuat DM, email, nomor telepon, atau identitas sensitif lainnya.

---

## Tujuan Penggunaan Dataset
Dataset ini **dibuat dan dipublikasikan hanya untuk tujuan berikut**:

- Penelitian akademik
- Analisis sentimen
- Text mining & Natural Language Processing (NLP)
- Evaluasi kebijakan publik secara ilmiah
- Replikasi dan validasi penelitian (research reproducibility)
